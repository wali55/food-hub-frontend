# Food Hub Frontend

This is the frontend part of the food ordering application. It has admin dashboard, provider dashboard and customer dashboard. From admin dashboard admin can see and manage all the users, can see orders and can manage categories. From provider dashboard provider can see and manage their orders and can manage their meals. From customer dashboard customer can see and manage their orders and can manage their profile. 

---

### Live URLs

- Frontend: https://food-hub-frontend-gamma.vercel.app/

---

### Features

- Customer can see meals, can order meals, make payment for the order via Paypal, can send review, can track order.
- Provider can see the manage meals, can manage orders.
- Admin can see and manage all the users, can see orders and can manage categories.

---

### Tech Stack

- Next.js
- Tailwind CSS
- Redux Toolkit
- Paypal

---

### Environment Variables

- BACKEND_URL: API URL
- NEXT_PUBLIC_PAYPAL_CLIENT_ID: PayPal Client ID

---

### How to Run

- Clone the repository
- npm install
- npm run dev
